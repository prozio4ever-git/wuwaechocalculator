中文操作说明：

副词条概率计算：在“副词条概率计算”页面，点击“栏位”选择你想要刷取的词条类型及数值下限，点击“一键精准计算”即可获取概率。

频整器概率计算：在“频整器概率计算”页面，勾选“锁定”以排除已有词条，并设置期望的副词条目标，点击“计算达成期望概率”即可查看刷取难度。

语言切换：点击页面右上角的语言菜单，可实时切换中文或英文界面。

English User Guide:

Sub-Stat Calculator: On the "Sub-Stat Calculator" tab, select the sub-stat type and minimum value for each slot, then click "Calculate Probability" to view the success rate.

Tuning Adjuster Calculator: On the "Tuning Adjuster Calculator" tab, check "Lock" to exclude existing sub-stats, set your desired targets for unlocked slots, and click "Calculate Tuning Odds" to evaluate your chances.

Language Switch: Click the language dropdown menu in the top-right corner to toggle between Chinese and English interfaces.
